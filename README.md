# 거래 내역 추적 웹 서비스, Fintrack
사용자의 거래(수입, 지출, 이체) 내역 관리 및 모니터링 기능과 자신의 거래 내역을 알림으로 받을 수 있는 기능을 제공하여 사용자가 효과적으로 자신의 거래 내역을 추적하는 데 도움을 주는 웹 서비스입니다.
